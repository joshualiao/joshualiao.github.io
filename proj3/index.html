Background

By interpolating both shape and color, you can create smooth facial morphs between faces.

Part 1 Correspondance

To get a nice facial morph, you need to manually annotate correspondance points between two images. By marking eyes, nose, mouth, and other key anchor points, individual parts of the face can be interpolated. This is done by separating the image into a triangular mesh, and interpolating each triangle. I used the provided correspondance tool here: https://cal-cs180.github.io/fa23/hw/proj3/tool.html.

Caricatures

Warps use the following equation: \( \alpha * im1 + (1 - \alpha) * im2 \). Typically, \( \alpha \) is between 0 and 1 to have a nice weighted average of two images. What happens if we increase it? Take \( \alpha = 2 \), which gives \( 2im1 -1 im2 \). We're subtracting the second image. Following from the idea of frequencies or sharpening, if we make im2 some kind of population mean, we can sharpen the features unique to the first image!
